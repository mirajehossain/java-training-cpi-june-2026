# Trainer Guide

1. Exception handling demo
2. Swing components form
3. Form validation demo
4. Basic calculator

## Day 2

1. JDBC insert demo
2. Login system
3. CRUD demo
4. Phonebook management system
5. Servlet overview

## Best teaching method

Show output first, then show code.

Do not teach Swing/JDBC as only theory. Teachers will understand better if every topic produces a working application.

## Must prepare before class

- JDK 17+
- Maven
- IntelliJ IDEA / NetBeans / Eclipse
- MySQL Server
- MySQL Workbench / phpMyAdmin
- Tomcat 10+
